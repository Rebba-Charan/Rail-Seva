# Rail-Seva 🚆

A console-based Railway Reservation System developed using **C++**, **Object-Oriented Programming**, **STL**, and **File Handling** concepts.

This project simulates a real-world railway reservation workflow including train management, ticket booking, passenger handling, and station administration.

---

# ✨ Features

## 👤 Passenger Module
- Passenger Login & Signup
- Search Trains
- Book Tickets
- Cancel Tickets
- View Tickets using PNR
- View & Edit Profile

---

## 🛠 RailAdmin Module
- Add Trains
- Cancel/Suspend Trains
- Manage Train Database
- Profile Management

---

## 🚉 Station Master Module
- Update Train Timing
- Update Train Dates
- Platform Announcements
- Profile Management

# ⚙️ How to Run

## Compile

```bash
g++ main.cpp src/*.cpp -o railseva
```

## Run

### Windows
```bash
.\railseva.exe
```

### Linux / Mac
```bash
./railseva
```
